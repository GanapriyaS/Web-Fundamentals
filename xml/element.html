XML Element

An XML element is everything from (including) the element's start tag to (including) the element's end tag.

An element can contain:
  • text - <title>, <author>, <year>, and <price>
  • attributes - <book> has an attribute (category="children")
  • other elements - <bookstore> and <book> have element contents, because they contain elements.
  • or a mix of the above
