XML XPath

  • XPath stands for XML Path Language
  • XPath uses "path like" syntax to identify and navigate nodes in an XML document
  • XPath is a major element in the XSLT standard.
  • XPath contains over 200 built-in functions.
  • In XPath, there are seven kinds of nodes: element, attribute, text, namespace, processing-instruction, comment, and document nodes.
  • XML documents are treated as trees of nodes. The topmost element of the tree is called the root element.
  • XPath is a query language for navigating in XML documents.
  • XPath is a W3C ( World wide web consortium )
  • To find nodes in XML document or elements in HTML web page.
  • XPath uses path expressions to select nodes or node-sets in an XML document. The node is selected by following a path or steps.

Note: If the path starts with a slash ( / ) it always represents an absolute path to an element!

S.No Expression Description
1 nodename Selects all nodes with the name "nodename"
2 / Selects from the root node
3 // Selects nodes in the document from the current node that match the selection no matter where they are
4 . Selects the current node
5 .. Selects the parent of the current node
6 @ Selects attributes
1 college Selects all nodes with the name "college"
2 /college Selects the root element college
3 college/student Selects all student elements that are children of college
4 //student Selects all student elements no matter where they are in the document
5 college//student Selects all student elements that are descendant of the college element, no matter where they are under the college element
6 //@lang Selects all attributes that are named lang

Absolute XPath - /college/student[1]/name

Relative XPath - //*/student[1]/name
