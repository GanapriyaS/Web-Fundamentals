FLEXBOX

  1. Flex direction is horizontal in defalult
  2. In its simplest use, you only specify the horizontal shadow and the vertical shadow
  3. flex-direction:row(default)/ column/ column-reverse/ row-reverse
  4. justify-content:center/ flex-start(default)/ flex-end
  5. align-items:flex-start(default),flex-end,center,stretch,baseline(in all center line among content)
  6. flex-flow: flex-direction flex-wrap;
  7. order1 - first , order6 - last , order default is 0
  8. default flex-grow and flex-shrink is 1
    • wrap avoid if contents exceed the container in row or col ,flex-wrap=wrap ,wrap-reverse,nowrap(default)
    • flex-grow default(1) - help to cover all remaining spaces i.e equal space 5- 5 times extra sapce taken
    • stop all not allow to sthrink 0 means not shrink
  9. flex-basis:100px; for flex-item - min or initial length of item
  10. flex:grow shrink basis; example: flex:5 1 100px;
