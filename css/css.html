CSS Cascading Style Sheets

  1. CSS is the language we use to style anHTML document.
  2. CSS describes how HTML elements. should be displayed.
  3. CSS saves a lot of work. It can control the layout of multiple web pages all at once
  4. A CSS rule-set consists of a selector and a declaration block:
    • The selector points to the HTML element you want to style.
    • The declaration block contains one or more declarations separated by semicolons.
    • Each declaration includes a CSS property name and a value, separated by a colon. Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

3 ways of inserting a style sheet

  • Inline CSS
  • Internal/Embedded CSS
  • External CSS
Inline CSS
  • An inline style may be used to apply a unique style for a single element.
  • To use inline styles, add the style attribute to the relevant element.
  • The style attribute can contain any CSS property.
<p style="color:red;background:orange">Hello<p>
Internal/Embedded CSS
  • An internal style sheet may be used if one single HTML page has a unique style.
  • The internal style is defined inside the <style> element, inside the head section.
&style>
.class_name{
color:red;
background:orange;
}
</style}
External CSS
  • An external style sheet is used to define the style for many HTML pages.
  • To use an external style sheet, add a link to it in the section of each HTML page.
