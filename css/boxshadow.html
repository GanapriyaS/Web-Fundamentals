BOX SHADOW

Add shadows to different < div > element

TEXT SHADOW EXAMPLES

Default
{ box-shadow: 5px 10px; }

A div element with a shadow. The first value is the horizontal offset a the second value is the vertical offset. The shadow color will be inherited from the text color.

Custom-color
{ box-shadow: 5px 10px #888888; }

You can also define the color of the shadow. Here the shadow color is grey.
