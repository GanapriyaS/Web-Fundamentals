FONT & TEXT MANIPULATION

  1. Priority pc>pt>px
  2. Absolute unit
    • px/mm/cm
  3. Relative unit
    • em depend on parent element
    • 1 vw or vh = one percent of body (total viewport)

EXAMPLES

Bold Italic Heading
{
font-family: 'PT Mono', monospace;
text-transform: capitalize;
text-align: center;
text-decoration:underline;
font-size: 1.08em;
font-weight: bold; /*font-weight:300;*/
font-style: oblique;
}

WEB FONTS

  • Web fonts allow Web designers to use fonts that are not installed on the user's computer.
  • When you have found/bought the font you wish to use, just include the font file on your web server, and it will be automatically downloaded to the user when needed.
  • Your "own" fonts are defined within the CSS @font-face rule.

In the @font-face rule

  1. first define a name for the font
  2. then point to the font file.

EXAMPLES

Web font - Pacifico style from google fonts
@font-face {
font-family: Font;
src: url("./font/Pacifico-Regular.ttf");
}

.font_web{
font-family: Font;
}
