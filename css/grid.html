GRID LAYOUT


  1. Grid-based layout system with rows and columns
  2. grid-template-rows and grid-template-columns => set rows and columns
  3. row-gap and column-gap for gaps between rows and columns
  4. repeat() function occupy all space
  5. grid-column-start and grid-column-end => depend on line number
  6. justify-items works on grid container and justify-self works on grid items
  7. place-items-center and place-self-center for both justify and align
  8. inline-grid not show space on right and left
  9. grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end
  10. Grid is focused on both width and height, while Flexbox is mainly focused on width.
