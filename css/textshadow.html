SHADOW

  1. Can add shadow to text and to elements The CSS text-shadow property applies shadow to text.
  2. In its simplest use, you only specify the horizontal shadow and the vertical shadow

TEXT SHADOW EXAMPLES

Default
{ text-shadow: 2px 2px; }
Custom-color
{ text-shadow: 2px 2px red; }
Blur Added
{ text-shadow: 2px 2px 5px red;}
White text with Added
{ color: white; text-shadow: 2px 2px 4px #000000;}
X-0 Y-0 no 3D
{ text-shadow: 0 0 3px #FF0000;}

MULTIPLE SHADOW EXAMPLES

Multiple Shadow

To add more than one shadow to the tex you can add a comma-separated list of shadow

{ text-shadow:0 0 3px #FF0000, 0 0 5px #0000FF; }
