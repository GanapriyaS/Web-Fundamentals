GRADIENTS

  1. CSS gradients let you display smooth transitions between two or more specifieu colors.
  2. CSS defines two types of gradients
    • Linear Gradients (goes down/up/left/right/ diagonally)
    • Radial Gradients (defined by their center)

CSS Linear Gradients

  1. To create a linear gradient you must def at least two color stops.
  2. Color stops are the colors you want to render smooth transitions among.
  3. You can also set a starting point and a direction (or an angle) along with the gradient effect.
  4. to top,to right,to left ,default(to bottom),to bottom-right,to-bottom-left, so on
top to bottom (Default)
{background-image: linear-gradient(red, yellow);}
left to right
{ background-image: linear-gradient(to right, red , yellow); }
top left to bottom right
{ background-image: linear-gradient(to bottom right, red, yellow); }

Using Angles

  1. If you want more control over the direct of the gradient, you can define an angle, instead of the predefined directions.
  2. A value of Odeg is equivalent to "to top".
  3. A value of 90deg is equivalent to "to right". A value of 180deg is equivalent to "to bottom".
0deg
{background-image: linear-gradient(red, yellow);}
90deg
{ background-image: linear-gradient(to right, red , yellow); }
180deg
{ background-image: linear-gradient(to bottom right, red, yellow); }
-90deg
{ background-image: linear-gradient(to bottom right, red, yellow); }

Multiple Color stops

Top to bottom
{background-image: linear-gradient(red, yellow, green);}
Rainbow background
{ background-image: linear-gradient(to right, red,orange,yellow,green,blue,indigo,violet); }

Using Transparency

  • CSS gradients also support transparency which can be used to create fading effect
  • To add transparency, we use the rgba() function to define the color stops.
  • The last parameter in the rgba() function can be a value from 0 to 1, and it defines the transparency of the color: 0 indicates full transparency, 1 indicates full color (no transparency).
Transparency
{background-image: linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1));}

Repeating a linear-gradient

The repeating-linear-gradient() function used to repeat linear gradients

A repeating gradient on 0deg axe starting red and finishing green
{background-image: repeating-linear-gradient(red, yellow 10%, green 20%);}
A repeating gradient on 45deg axe starting red and finishing green
background-image: repeating-linear-gradient(45deg,red,yellow 7%,green 10%);
A repeating gradient on 190deg axe starting red and finishing green
background-image: repeating-linear-gradient(190deg,red,yellow 7%,green 10%);
A repeating gradient on 90deg axe starting red and finishing green
background-image: repeating-linear-gradient(90deg,red,yellow 7%,green 10%);

CSS Radial Gradients

  1. By default, shape is ellipse, size is farthest-corner, and position is center.
  2. A radial gradient is defined by its center.
  3. The shape parameter defines the shape. It can take the value circle or ellipse. The default value is ellipse.
Evenly Spaced Color Stops (this is default)
background-image: radial-gradient(red, yellow, green);
Differently Spaced Color Stops
{ background-image: radial-gradient(red 5%, yellow 15%, green 60%); }
Shape is circle
{ background-image: radial-gradient(circle, red, yellow, green); }
Shape is circle with different stop
{ background: radial-gradient(circle,white 40%,rgba(255, 203, 31,0.9) 50%,pink 30%); }
repeating-radial-gradient() function is used to repeat radial gradients
{ background-image: repeating-radial-gradient(red, yellow 10%, green 15%); }
