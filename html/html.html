Web Technology

Web technology is defined as the technology through which we can develop or create various types of web pages.

Web page

A document which can be displayed in a web browser such as Firefox, Google Chrome, Opera, Microsoft Internet Explorer, etc.

Website

A collection of web pages which are grouped together and usually connected together int various ways.

Web Server

A computer that hosts a website on the Internet.

Web Browser

A web browser is a software application for accessing information on the World Wide Web.

Page Structure Elements

  • < header > - Defines a header for a document or a section
  • < nav > - Defines a set of navigation links
  • < section > - Defines a section in a document
  • < article > - Defines an independent, self-contained content
  • < aside > - Defines content aside from the content (like a sidebar)
  • < footer > - Defines a footer for a document or a section
  • < details > - Defines additional details that the user can open and close on demand
  • < summary > - Defines a heading for the < details > element

<html>

  • The <html> tag represents the root of an HTML document.
  • The <html> tag is the container for all other HTML elements.

<!DOCTYPE>

  • The declaration is not an HTML tag.
  • It is an "information" to the browser about what document type to expect.

How are web pages accessed?

When a user requests a web page from a particular website, the web browser retrieves the necessary content from a web server and then displays the page on the screen.

How are Web pages created?

Web pages are created using Markup Languages.

HTML is the standard markup language for creating Web pages.

HTML

  • HTML stands for Hyper Text Markup Language
  • HTML describes the structure of a Web page
  • HTML elements tell the browser how to display the content
