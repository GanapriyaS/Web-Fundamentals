< head >

  1. The < head > element is a container for metadata (data about data)
  2. Placed between the < html > tag and the < body > tag.
  3. Metadata is data about the HTML document. Metadata is not displayed.
  4. Metadata typically define the document title, character set, styles, scripts, and other meta information.
  5. The following elements can go inside the < head > element: 
                <title></title>
            <style></style>
            <base>
            <link>
            <meta>
            <script></script>
            <noscript></noscript>
