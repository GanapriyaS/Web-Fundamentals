JS Date Objects



Method Description
getFullYear() Get the year as a four digit number (yyyy)
getMonth() Get the month as a number (0-11)
getDate() Get the day as a number (1-31)
getHours() Get the hour (0-23)
getMinutes() Get the minute (0-59)
getSeconds() Get the second (0-59)
getMilliseconds() Get the millisecond (0-999)
getTime() Get the time - no of milliseconds (milliseconds since midnight January 1, 1970)
getDay() Get the weekday as a number (0-6)
Date.now() Get the time. ECMAScript 5.
Creating Data Objects
  • Data Objects are created with the new Date() constructor.
  • 4 ways to create a new date object:
    • new Date()
    • new Date(year,month,day,hours,minutes,seconds,milliseconds)
    • new Date(milliseconds)
    • new Date(date string)
new Date()

Creates a new date object with the current date and time

new Date(year, month, ...) => new Date(2018,11,24,10,33,30,0) creates a new date object with a specified date and time. 7 numbers specify year, month, day, hour, minute, second, and millisecond (in that order):

new Date(milliseconds) creates a new date object as zero time plus milliseconds: 01 January 1970, 00:00:00 Universal Time(UTC) plus 100 000 000 000 milliseconds is approximately 03 March 1973:

January 01 1970 minus 100 000 000 000 milliseconds is approximately October 31 1966:

Note: One day(24 hours) is 86,400,000 milliseconds.

Date String Formats

Short Dates

Short dates are written with an "MM/DD/YYYY"

const d = new Date("03/25/2015");
Long Dates

Long dates are most often written with a "MMM DD YYYY"

const d = new Date("Mar 25 2015");
